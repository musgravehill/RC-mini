Прошивка устройств через S.Port c помощью компьютера

Для обновления прошивки S.Port устройств можно исопльзовать компьютер под управлением MS Windows и соответствующий адаптер (например, FrSky STK или его аналог на Arduino). Для этого следует загрузить программу FrSky SPORT upgrade adapter.zip. Распакуйте архив в отдельную папку.


Схема подключения

Схема подключения может несколько отличаться в зависимости от используемого адаптера. В случае STK на базе Arduino Nano, схема выглядит следующим образом:

Джампер "Upgrade mode" должен быть установлен (т.е. должен замыкать два контакта, обозначенных серым цветом в верхней части картинки).

Обратите внимание: для прошивки необходимо, чтобы устройство, в которое загружается прошивка, вошло в режим "загрузчика" (bootloader). Устройства FrSky могут быть переключены в этот режим только в момент подачи питания. Поэтому подключение прошиваемого устройства должно проводиться строго в соответствии с процедурой (см. ниже), и не ранее!
